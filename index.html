Core 1: Interaction

FALL 2024

Core 1: Interaction is designed to introduce students to programming as a creative medium—as a way of making and exploring. The coursework focuses on developing a vocabulary of interaction design principles which can then be applied across a range of platforms.

Assignments

  1. Fill out the intro survey if you haven't already.
  2. Download Visual Studio Code and Figma, if you weren't able to in class
  3. Read and respond: Read What Is Code and add 2-3 sentences, a quote, an image, or a thought to this Are.na board
  4. Come to class with 3 ideas for your Harmonic Collection’s theme. Think of these potential themes as the overarching concept you’ll explore through a series of sketches. Make a lo-fi presentation using artboards in Figma and add to the Harmonic Collection Ideas project.
